# 📖 A Story: Guest 666 (Interactive Web Story)
로블록스의 유명한 괴담(Myth) 중 하나인 **'Guest 666'**을 새로운 시각으로 재해석한 인터랙티브 웹 스토리입니다.
단순한 공포 괴담이 아닌, 그가 왜 붉은 스크립트에 타락하게 되었는지에 대한 슬프고 감동적인 우정 이야기를 담았습니다.
화면을 클릭하며 이야기를 진행해 보세요.
## ✨ 주요 기능 (Features)
- 🖱️ **인터랙티브 진행:** 화면을 클릭할 때마다 다음 문장이 자연스럽게 나타납니다.
- ✍️ **타이핑 효과:** 몰입감을 높여주는 텍스트 타이핑 애니메이션.
- 🎵 **배경음악(BGM):** 스토리에 어울리는 어둡고 감성적인 음악 재생.
- 🌍 **다국어 지원:** 한국어(Korean) 및 영어(English) 언어 선택 기능.
- 📱 **반응형 웹:** 모바일과 데스크톱 모두에서 쾌적하게 읽을 수 있는 세련된 다크 테마 UI.
## 🛠️ 사용된 기술 (Tech Stack)
- HTML5
- CSS3 (Custom Variables, Flexbox, Animations)
- Vanilla JavaScript
